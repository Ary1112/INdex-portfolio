Personal Portfolio Page Development

Author: Aaryan Kushwaha

Date: 2025-05-17

Overview

This report outlines the process of creating my personal portfolio page using only HTML. The aim was to build a clean, accessible, and content-driven website without applying any external CSS.

Structure

The layout is built using HTML5 semantic elements:

  • <header>: Displays my name, a short tagline, and navigation links to other sections.
  • <main>: Comprises three major parts:
    • <section id="about">: A brief personal introduction under "About Me".
    • <section id="projects">: Describes three highlighted projects with links and summaries.
    • <section id="contact">: Contains contact information and social media links.
  • <footer>: Provides copyright and footer content.

Features

  • Used semantic HTML5 tags like <header>, <main>, <section>, and <footer> to enhance readability and structure.
  • Internal navigation enables smooth section jumping.
  • Projects are presented using descriptive <article> elements.
  • Interactive email and social media links offer easy communication.

Learning

This exercise deepened my understanding of HTML structure and semantics. By deliberately excluding CSS, I focused more on meaningful layout, proper tag usage, and user accessibility.
